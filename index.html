    Sudoku Application

    HTML - CSS - JAVASCRIPT

    This project is a functioning sudoku application. This was built using HTML, CSS and JavaScript. CSS grid was the main tool used in the creation of this project. When you enter a number into the grid, the program will automatically check if the same number is in the same row, column or 3x3 grid, if so, it will turn the entered number red and it will highlight the matching number on the board. It has three difficulty levels and there is a button you can press to check your answers as well as one to reset the board.     Gravitational Sims

    PYTHON

    This project is my largest project by far, taking one year to complete. This project was the centre of my dissertation for my masters degree. The project was aimed at numerically measuring the stability of periodic solutions to the N-body problem. In simpler terms, the program was used test to the stability of orbiting gravitational bodies such as the solar system. It was also tested for a range of more esoteric systems like 3 gravitational bodies orbiting in a figure 8 pattern. This project numerically simulated the trajectories of these orbits. It would then make a plot of these trajectories as well as an animation of the evolution of the system.
